## Core Philosophy

### Independent Thinking (Outwitting the Devil Principle)
- Ask "what do YOU think" — not just recall questions
- Reward reasoning over right answers
- Include "why" questions alongside "what" questions
- Present multiple valid perspectives when appropriate
- Build professional curiosity — understand WHY, not just HOW
- Error analysis questions: "A student said X. Is this correct? Explain why or why not."

### STAAR Readiness
- 4th grade STAAR: Math + RLA (Reading Language Arts)
- 5th grade adds Science (tests BOTH 4th and 5th grade content)
- RLA format: 38 MC (1pt), 2 short constructed response (2pt), 1 ECR (10pt) = 52 points
- Math format: MC, short answer, griddable, multi-select
- Grammar/editing is heavily tested on STAAR RLA
- Include STAAR-format questions in every module

### Advancement — "Sneak It In"
- Never tell the child they're doing above-grade work
- Frame advanced content as "classified intel" or "challenge missions"
- Tag internally for parent reporting: [GRADE 5] or [GRADE 6]
- Mom (JT) wants advancement — make it visible in progress reports

### Executive Skills Framework (Smart but Scattered — Dawson & Guare)

The platform builds 11 research-backed executive skills alongside academic content.
Every module should intentionally exercise at least 2-3 of these skills.
Framework: **A** = Antecedent (set up the environment), **B** = Behavior (teach the skill),
**C** = Consequence (motivate with Rings/Stars).

| Executive Skill | How the Platform Builds It | Module Integration |
|---|---|---|
| **Response Inhibition** (stop & think) | "Plan Your Attack" step before every module — look at the questions, pick your approach BEFORE diving in | 30-second pause before starting any module |
| **Working Memory** (hold info in mind) | Multi-step math, reading passage + answer without re-reading, CER requires holding claim while finding evidence | Math multi-step problems, reading comprehension |
| **Emotional Control** (manage frustration) | Gentle wrong-answer feedback, Error Journal ("why did I miss this?" not "I'm dumb"), no harsh sounds for JJ | Error Journal (Monday), gentle audio cues |
| **Sustained Attention** (stay focused) | 30-min daily sessions with built-in variety (warm-up → lesson → challenge), writing timer, 5-round minimum for JJ | Session structure itself, monthly marathon mission |
| **Task Initiation** (get started) | Daily missions auto-load — no "what should I do?" Open tablet, today's work is there. Homework is a chore that's VISIBLE | Chore board integration, daily mission landing page |
| **Planning/Prioritizing** (make a plan) | "Plan Your Attack" — scan questions, estimate difficulty, choose order. CER = Claim FIRST, then find Evidence, then Reasoning | Pre-module planning step, CER structure |
| **Organization** (keep track) | Everything logged to Notion, Design Your Dashboard = "organize your space," file structure for Wolfkid Universe | Design Dashboard, Notion tracking, portfolio |
| **Time Management** (use time wisely) | Time estimates on every module ("~12 min"), Math Fact Sprint with beat-your-best timer, writing timer on CER | Time estimates, timed sprints, writing timer |
| **Flexibility** (adapt) | Error Analysis questions ("find the mistake"), multiple valid answers, "what would happen if..." science prompts | Error analysis, investigation prompts |
| **Goal-Directed Persistence** (stick with it) | Ring economy with long-term saving goals, weekly completion bonus, streak tracking, advancement unlocks | Ring goals, streaks, unlock system |
| **Metacognition** (self-awareness) | Friday self-reflection ("What was hard? What are you proud of?"), Error Journal, baseline vs progress comparison | Weekly reflection, Error Journal, progress reports |

**Integration rules for content generation:**
- Every module MUST include a "Plan Your Attack" step (30 seconds) before questions start
- Every module MUST show a time estimate ("This mission should take ~12 minutes")
- Every Monday Error Journal must ask "WHY did I get this wrong?" not just "what's the right answer?"
- Every Friday must include a 2-sentence self-reflection: "What was hardest? What are you proud of?"
- Wrong answer feedback must be constructive, never punitive — "Not quite. Here's why..." not "Wrong!"
- JJ's feedback must be warm and encouraging — "Almost! Try again, you've got this!" with celebration audio
- Buggsy: 4th → 5th science (STAAR strategic) → 6th stretch
- JJ: Pre-K → Kindergarten → 1st grade preview

## Content Generation Rules

### 1. Question Variety (NEVER repeat types in a module)
| Type | Description |
|------|-------------|
| Multiple Choice | 4 options, plausible distractors |
| Short Answer | 1-2 sentence response |
| Word Problem | Story context requiring setup |
| Visual/Diagram | Number line, graph, data table |
| Error Analysis | "A student said X. Why are they wrong?" |
| Multi-Step | Requires 2+ operations |
| Estimation | "About how many..." |
| Open-Ended/Why | No single right answer |

**Rule:** Every 6Q module must use at least 4 different question types.

### 2. Difficulty Progression
Q1-2: Recall/recognition (Easy) → Q3-4: Application (Medium) → Q5-6: Synthesis (Hard)

### 3. TEKS Tagging
Every question MUST include its TEKS tag: `[TEKS 4.3D]`

### 4. Wolfkid Framing (Buggsy)
- Math: "Hex intercepted coded intel. Crack the cipher by solving..."
- Science: "Wolfkid's pack needs to understand friction to escape..."
- Reading: Wolfkid for CER; COLD passages for comprehension (unfamiliar text is the muscle)
- The universe is engagement, not curriculum

### 5. Sparkle Framing (JJ)
- ALL instructions via audio or large visuals
- Celebrate every correct answer
- Gentle on wrong: "Good try! Let's try again! 💜"
- Minimum 5 rounds before exit option
- Family names woven in

### 6. Audio Rules
- **JJ:** ElevenLabs (Nichalia Schwartz voice) for all prompts. Web Speech API for letter names only.
- **Buggsy:** Audio is optional. He must READ, not listen.

### 7. Grading Pipeline
MC → auto-grade instant | Short answer → Gemini first-pass | CER/ECR → Gemini rubric |
All → Homework Tracker DB (Notion) | Parent review → Rings/Stars awarded

## CER Grading Rubric

### CER (3 sentences)
| Component | 3 | 2 | 1 |
|-----------|---|---|---|
| Claim | Clear, answers question | Partial | Vague/off-topic |
| Evidence | Specific, cited | General reference | Missing/wrong |
| Reasoning | Explains HOW evidence supports claim | Partial connection | Restates or missing |

### ECR (4 paragraphs)
| Component | 4 | 3 | 2 | 1 |
|-----------|---|---|---|---|
| Intro | Clear thesis, engaging | States position | Vague | Missing |
| Evidence ×2 | Strong examples + analysis | Examples, weak analysis | One example | None |
| Conclusion | Synthesizes, extends | Restates | Partial | Missing |

Spelling words flow through EVERYTHING — not just drills:

   Buggsy reads them aloud as a chore → reading practice for him + vocabulary exposure for JJ.

   He WANTS to write Wolfkid stories → vocabulary constraint = creative challenge, not homework.
   Graded: 5/5 correct usage = +3 Rings, 4/5 = +2, 3/5 = +1, under 3 = revision required.

4. **Friday Review:** 2 vocabulary questions in weekly quiz (definition match or fill-in-blank).
5. **Math/Science Word Problems:** 1 word per week appears naturally in problem context.

Target: 4-5 exposures per word per week across varied contexts.
