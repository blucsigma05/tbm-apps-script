# Texas TEKS — 4th Grade Math Standards (Adopted 2012, Current)

## Overview
Grade 4 math focuses on: place value through billions, operations with whole numbers,
fractions, decimals (tenths/hundredths), geometry, measurement, data analysis,
and personal financial literacy. Students should NOT use calculators.

---

## Strand 1: Number and Operations — Whole Numbers (4.2, 4.4)

### 4.2 — Place Value and Comparisons
- Read, write, compare, order whole numbers to 1,000,000,000
- Represent numbers on number lines
- Round to nearest 10, 100, 1,000
- Determine value of digits (ones through hundred millions)

### 4.4 — Whole Number Operations
- Add and subtract whole numbers (with regrouping)
- Multiply up to 4 digits × 1 digit, 2 digits × 2 digits
- Divide up to 4-digit dividends by 1-digit divisors (with remainders)
- Use strip diagrams and equations to represent problems
- Estimation to determine reasonableness

**Variety contexts:** Money, distance, population, sports stats, cooking quantities, building

---

## Strand 2: Number and Operations — Fractions (4.3)

### 4.3A-F — Fractions
- Represent fractions (unit fractions, equivalent fractions)
- Compare fractions with different denominators (benchmarks 0, 1/2, 1)
- Decompose fractions: 3/4 = 1/4 + 1/4 + 1/4
- Add and subtract fractions with like denominators
- Represent fractions and decimals on number lines
- Relate decimals to fractions (tenths, hundredths)

**Variety contexts:** Pizza/food sharing, measuring ingredients, sports statistics, time

---

## Strand 3: Number and Operations — Decimals (4.2, 4.4)

### Decimal Operations
- Represent decimals (tenths, hundredths) including money
- Compare and order decimals using symbols
- Relate decimals to fractions
- Add and subtract decimals (through hundredths)

**Variety contexts:** Money/prices, measurements, batting averages, temperature

---

## Strand 4: Algebraic Reasoning (4.5)

### 4.5 — Patterns and Relationships
- Represent multi-step problems with equations using a variable
- Generate number patterns using addition, subtraction, multiplication, division
- Use input-output tables
- Represent problems using strip diagrams

**Variety contexts:** Patterns in nature, savings over time, game scoring, building sequences

---

## Strand 5: Geometry and Measurement (4.6, 4.7, 4.8)

### 4.6 — Geometry
- Identify points, lines, line segments, rays, angles, parallel/perpendicular lines
- Classify 2D figures by attributes (sides, angles, symmetry)
- Lines of symmetry

### 4.7 — Measurement
- Convert measurements within the same system (customary and metric)
- Inches ↔ feet ↔ yards, cups ↔ pints ↔ quarts ↔ gallons
- Centimeters ↔ meters ↔ kilometers, milliliters ↔ liters
- Solve problems involving measurement conversions

### 4.8 — Area and Perimeter
- Find perimeter and area of rectangles
- Use formulas: P = 2l + 2w, A = l × w
- Solve real-world problems with area and perimeter

**Variety contexts:** Room design, garden planning, sports fields, wrapping presents, maps

---

## Strand 6: Data Analysis (4.9)

### 4.9 — Data Representation
- Represent data with frequency tables, dot plots, stem-and-leaf plots
- Solve problems using data from tables and graphs
- Interpret data to draw conclusions

**Variety contexts:** Class surveys, weather data, sports statistics, science experiments

---

## Strand 7: Personal Financial Literacy (4.10)

### 4.10 — Financial Literacy
- Distinguish between fixed and variable expenses
- Calculate profit (income - expenses)
- Describe advantages/disadvantages of saving and spending
- Understand the concept of financial institutions (banks, credit unions)

**Variety contexts:** Lemonade stand, allowance budgeting (connects to Kids Hub Rings!),
saving for a goal, business decisions

## Question Type Rotation Guide

To ensure variety, rotate through these question formats:

| Type | Description | Example |
|------|-------------|---------|
| Computation | Straight calculation | 4,328 × 7 = ? |
| Word Problem | Story context requiring setup | "If Nathan has 156 comics and gives 1/4 to a friend..." |
| Visual | Number line, diagram, model | "Place 3/8 on this number line" |
| Error Analysis | Find and explain a mistake | "A student said 1/3 > 1/2. Is this correct? Explain." |
| Multi-Step | Requires 2+ operations | "Buy 3 items, calculate change, split remainder" |
| Estimation | Reasonableness check | "About how many...?" |
| Real-World | Applied mathematics | "Your room is 12 ft × 10 ft. How much carpet do you need?" |
| Pattern | Identify or extend a pattern | "What's the rule? 3, 7, 11, 15, ..." |

**Rule:** Never use the same question type for more than 2 consecutive questions in a module.
**Rule:** Each module must include at least 3 different question types.
